Opdracht 3

Waarom staat de tekst in de blokken horizontaal gecentreerd?

Omdat de text-align property op center staat in het css.

Waarom heeft het vertical-align: center geen effect

Omdat het geen effect heeft op de text, het is bedoeld voor een ander soort elementen

Wat doet vertical-align?

Zoals je kan zien bij vertical-align: middle, bij deze zorgt hij ervoor dat het midden van je div elementen allemaal op eenzelfde lijn komen.

Dus de vertical-align property zorgt ervoor dat je al je elementen op eenzelfde hoogte kan krijgen.

Wat is de simpele manier om een enkele regel tekst verticaal te centreren?

Aan de hand van de line-height property en dan een gegeven waarde kan je ze centreren, maar als je dit op 1 element wil dan moet je dat element selecteren van bijvoorbeeld het nth-child property.

Welke truc wordt gebruikt om meerdere regels verticaal te centreren?

Ze combineren display:table-cell samen met vertical-align:middle.
